# हस्तBoli - HastBoli
HastBoli is a learning management platform for a course that teaches Indian Sign Language.
## Getting Started

```
git clone https://github.com/ryzbaka/HandyMan.git
cd HandyMan
node server.js
```
### Prerequisites
* Node.js Version: 10.16.3
* NPM Version: 6.9.0
```
npm install express
```
